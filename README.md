# Dockerized .NET Web API + MicrosoftSQL (EFCore)

## Prerequisites
- Docker Desktop
- .NET SDK
- Clone Repo

## Build API Container
```bash
docker-compose up -d --build
```

## Description
```
Database first approach
Swagger UI
CRUD Operations for Customers, Products, and orders.
Reusable data layer
Couple Tests (Could be improved and expanded)
```
